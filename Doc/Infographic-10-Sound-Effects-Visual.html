Visual Effects System (10 Effect Types)
💥

1. Screen Shake

triggerShake()

CSS transform with random jitter applied to the game container. Creates an impact feel on goals, explosions, and blinker moments.

300-500ms CSS transform random offset

2. Screen Flash

Color overlay — full screen

Color overlay covering the entire screen. Green for goals, yellow for saves, red for misses, red pulse for blinkers.

200-400ms opacity fade goal save miss

3. Confetti Particles

30-50 particles · physics simulation

Array of colored particles with gravity, wind, and rotation. Uses the C palette: cyan, gold, pink, purple, orange. Triggered on wins, achievements, and perfect shots.

gravity + wind rotation C palette

4. Smoke Particles

Cloud particles · opacity fade + scale

Cloud-like white and gray particles that rise upward. Opacity fades out while scale increases, creating an expanding smoke effect during puff animations.

opacity fade scale increase puff trigger

5. Puff Bubbles

Floating circles · drift + rise

Floating circular bubbles during the puff charge phase. They rise upward with a slight horizontal drift, varying from small to medium sizes.

vertical rise horizontal drift size variation

6. Audience Bubbles

Spectator emoji reactions

Random spectator emoji reactions that float up from the bottom of the screen. Used during exciting gameplay moments to simulate crowd engagement.

emoji particles float upward random spawn

7. Puff Wave

Radial wave · center outward

Full-screen radial wave effect triggered by the Puff Wave audience action. A ripple animation expands from the center outward across the entire screen.

radial expand full-screen audience action

9. Dim Lights

Screen darken · focus effect

Screen darkens during the puff charge phase, creating a focus effect that draws attention to the power meter. Returns to normal brightness on release.
