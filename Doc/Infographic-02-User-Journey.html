5

Back Navigation System

window.__moodlabGoBack() — Smart Step-Back

Priority-ordered stack. The back button lives in index.html outside the React DOM.

Game overlay ➡ Close game (remove overlay)
Matchmaking ➡ Cancel matchmaking
Game selected ➡ Deselect game card
Zone hub ➡ Return to Arena Hub (lobby)
Arena hub ➡ Stays (no further back)
💡

Why outside React? Game overlays use position: fixed which creates new stacking contexts. A back button inside React would be trapped beneath the overlay. Placing it in index.html at the top z-index layer ensures it always remains accessible.

// index.html — outside React root
function __moodlabGoBack() {
  if (gameOverlayOpen)    return closeGame();
  if (matchmaking)       return cancelMatch();
  if (gameSelected)     return deselectGame();
  if (inZoneHub)        return goToArena();
  // arena hub — stay
}

Walking Transition

Smooth Zone Navigation Animation

When navigating between zones, a walking animation plays to give a sense of movement through the arena. A walking state flag prevents double-navigation during the transition.

Zone A
current
🚶
walking = true

destination
⚠️

Double-nav guard: While isWalking === true, all zone navigation taps are ignored. This prevents race conditions where rapid tapping could cause partial renders or broken state.
