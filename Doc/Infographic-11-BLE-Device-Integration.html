MOOD LAB ARENA — BLE Device Integration

Cannabis vape device as game controller · Web Bluetooth API · Real-time puff measurement

01 — The Concept

Vape Device = Game Controller

Mood Lab Arena transforms a cannabis vape device into a game controller. Connected via the Web Bluetooth API (BLE), puff duration is measured in real-time and mapped to in-game actions. Partnership with Cali Clear vape brand.

🎮

Game Controller

Vape device sends puff data that drives gameplay mechanics
📡

Web Bluetooth API

BLE connection via browser-native GATT protocol
⏱️

Real-Time Measurement

Puff duration captured in milliseconds, normalized to power %
🌿

Cali Clear Partner

Official hardware partner with Standard and Select device lines

04 — BLE Connection Flow

Connection & Calibration Sequence

1

User Taps "Connect Device"

showBlePopup = true — BLE connection modal is displayed

2

Browser BLE Scan Dialog

Native browser Bluetooth device picker is triggered. User selects their Cali Clear device.

3

Device Connected

bleConnected = true — GATT server connection established.

4

Device Calibration

showDeviceOptimize = true — 4-stage calibration from 0-100%.

Stage 0: "Detecting device..."
Stage 1: "Calibrating sensors..."
Stage 2: "Optimizing response..."
Stage 3: "Testing connection..."
5

Session Input Configured

sessionInput set based on device capabilities. Input type selector is available.

✓

Ready to Play!

Device is calibrated and input mode is set. User can launch any compatible game.

How Input Selection Works

07 — Session Input

Persistent Input State

🔄

sessionInput State

  • No repeated device selection prompts per game

10 — Fallback Behavior

Graceful Degradation

The system never blocks gameplay. Every failure state falls back to tap mode so users can always play.

📵

No BLE Device

→ Tap mode (Open pool, 1× rewards)

🔌

BLE Disconnects Mid-Game

→ Auto-switch to tap, game continues seamlessly

🌐

Browser Doesn't Support BLE

→ Tap-only mode, no BLE UI elements shown
